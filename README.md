### Installation

1. Clone the repository
   ```sh
   git clone https://github.com/razifrosadi/rosadi-restaurant-app.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Build Project
   ```sh
   npm run build
   ```
   
      ```sh
   npm run build-image
   ```
   
4. Run Project
   ```sh
   npm run start-dev
   ```
